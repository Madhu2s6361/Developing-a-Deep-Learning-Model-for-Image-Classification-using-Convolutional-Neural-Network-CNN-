# Breast Ultrasound Image Classification (CNN) 
What it is:

Breast ultrasound imaging is used to examine breast tissues and detect abnormalities like lumps or tumors. It is widely used because it is safe, non-invasive, and effective even in dense breasts (where mammography is not as clear).

Why this is important ?
1. Helps detect breast cancer early
2. Assists radiologists in diagnosis
3. Reduces human error in examining images
4. Useful for screening in hospitals and rural areas

Goal of the to build a CNN model that can classify breast ultrasound images into:
1. Normal
2. Benign (non-cancerous)
3. Malignant (cancerous)
The model learns patterns from ultrasound textures and can predict the type of lesion.
